# Imagen Prompt Engineering Guide

## Prompt Structure

A good prompt has three components:
1. **Subject** - What is the main focus?
2. **Context/Background** - Where is it placed?
3. **Style** - What artistic style?

### Example Evolution

| Level | Prompt |
|-------|--------|
| Basic | "A park in spring next to a lake" |
| Better | "A park in spring next to a lake, the sun sets across the lake, golden hour" |
| Best | "A park in spring next to a lake, the sun sets across the lake, golden hour, red wildflowers, photorealistic" |

## Aspect Ratio Guide

| Ratio | Best For | Example Prompt |
|-------|----------|----------------|
| 1:1 | Social media, Instagram | Product shots, portraits |
| 4:3 | Photography, screens | Landscapes, general photos |
| 3:4 | Portrait orientation | Portraits, full-body shots |
| 16:9 | Widescreen, video | Landscapes, cinematic |
| 9:16 | Vertical video | Mobile content, tall subjects |

## Common Mistakes to Avoid

1. **Too vague** - "A nice landscape" → "A mountain landscape at sunset with dramatic clouds"
2. **Conflicting styles** - Mixing photorealistic with cartoon in one prompt
3. **Overly complex** - Too many competing elements
4. **Ignoring lighting** - Always specify lighting for better results
5. **Forgetting aspect ratio** - Use appropriate ratio for your subject

## Iterative Refinement

Start simple, then add details:
1. Core subject
2. Setting/context
3. Lighting
4. Style
5. Technical modifiers (lens, camera)
6. Quality modifiers

Example progression:
```
Step 1: "A cat"
Step 2: "A cat sitting on a windowsill"
Step 3: "A cat sitting on a windowsill, afternoon sunlight"
Step 4: "A cat sitting on a windowsill, afternoon sunlight streaming through, cozy atmosphere"
Step 5: "A cat sitting on a windowsill, afternoon sunlight streaming through, cozy atmosphere, 50mm lens, soft focus, warm tones"
```
