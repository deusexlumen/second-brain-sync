# Bild-Generierung Skill

Generiert Bilder via maxm-imggenurl.web.val.run API.

## Usage

```
!bild <prompt>
!bild <prompt> --enhance
!bild <prompt> -e
```

## Options

- `--enhance` oder `-e`: Verbessert den Prompt automatisch mit zusätzlichen Details (Licht, Stil, Qualität)

## Example

```
!bild a cyberpunk city at night with neon lights
!bild a cyberpunk city --enhance
```

## Output Format

Returns Markdown mit eingebettetem Bild und kurzem Kommentar.
