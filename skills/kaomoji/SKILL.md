# Skill: kaomoji

Zeigt ein zufälliges Kaomoji oder sucht nach einem bestimmten Tag.

## Usage

```
/kaomoji
/kaomoji evil
/kaomoji hug
/kaomoji heart
```

## Tags

Verfügbare Tags zum Suchen:
- `angel` - Himmlische/Engels-Kaomojis
- `anger` - Wütende/Kämpferische Kaomojis  
- `annoyed` - Genervte Kaomojis
- `blush` - Errötende/Schüchterne Kaomojis
- `crying` - Weinende/Traurige Kaomojis
- `dancing` - Tanzende Kaomojis
- `devil` - Teuflische/Böse Kaomojis
- `excited` - Aufgeregte/Energetische Kaomojis
- `fight` - Kampf-Bereite Kaomojis
- `heart` - Herzen/Liebe Kaomojis
- `hug` - Umarmungs-Kaomojis
- `kiss` - Kuss-Kaomojis
- `sad` - Traurige Kaomojis
- `smiling` - Lächelnde Kaomojis
- `smirk` - Schmunzelnde Kaomojis
- `sparkles` - Glitzernde/Magische Kaomojis
- `surprised` - Überraschte Kaomojis
- `table_flip` - Tisch-Umwurf-Kaomojis
- `wink` - Zwinkernde Kaomojis

## Examples

**Zufällig:**
```
(◍•ᴗ•◍)♡ ✧*。
*— Good energy.*
```

**Gefiltert (`/kaomoji devil`):**
```
ψ(｀∇´)ψ
*— Chaos incarnate.*
```

Nutzt das [emoticon_kaomoji_dataset](https://github.com/ekohrt/emoticon_kaomoji_dataset) mit über 62.000 Kaomojis.
