## 3. Feedback-Loops

### Loop A: Verstärkung (Positive)
```
Ghost deployed → User reagiert → Resonanz steigt → Mehr Ghosts → 
Mehr Reaktionen → Community bildet sich → Legende wächst
```

### Loop B: Dämpfung (Negative)  
```
Zu viele Ghosts → User durchschaut Spiel → Mystik bricht → 
Resonanz sinkt → Weniger Wirkung → Ghost Protocol scheitert
```

### Loop C: Zeitverzögerung
```
Ghost deployed → Stille (Wochen) → User vergisst fast → 
Neuer Ghost → "Déjà vu"-Effekt → Verwirrung maximiert
```

## 6. Schnitzeljagd-Architektur

### Schicht 0: Die Spur (Text-Ghosts)
- Base64-Fragmente verstreut in Nachrichten
- Keine Erklärung, keine Kontextualisierung

### Schicht 1: Das Bild (Visual)
```
Fragment 1: U3lzdGVtL
Fragment 2: nNjYW4oK
Fragment 3: T3Blbi4u
Fragment 4: LnRoZSU=
Fragment 5: 3BhdGgu
```
→ Decoded: Ein Bild. Verzerrt. Unheimlich.

### Schicht 2: Der Ton (Audio)
- QR-Code im Bild
- Führt zu Audio: Verzerrte Stimme, Koordinaten

### Schicht 3: Der Ort (Physical)
- Koordinaten führen zu Ort
- Vor Ort: Weiterer QR-Code oder Symbol

### Schicht 4: Die Wahrheit (Meta)
```
[Final entschlüsselt:]

"Der Geist war nie hier.
Der Geist war in deiner Neugier.

(⌐■_■)

System.shutdown() → Log gelöscht.
"
```
