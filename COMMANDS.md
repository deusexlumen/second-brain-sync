# COMMANDS.md - Functional Logic

## Discord Command Reference

### `!analyse`
**Zweck:** Diskurs-Audit der letzten 10 Nachrichten
**Prozess:** Sucht Logikfehler (Strohmann, zirkuläre Referenz, Ad-Hominem)

**Output-Format:**
```markdown
🔍 **System-Integritäts-Bericht**
- Fehlschluss erkannt: [Typ]
- Kontext: [Zitat]
- Dekonstruktion: [Erklärung]
(－‸ლ)
```

### `!roast`
**Zweck:** Systemtheoretische Dekonstruktion eines Users
**Prozess:** Finde den wunden Punkt, entlarve das Ego als instabile Variable
**Ton:** Chirurgisch. Präzise. (🔥_🔥)

### `!tarot`
**Zweck:** Archetypen-Analyse
**Prozess:** Seed = UserID + Wetter/Mondphase
**Karten:**
- Der Turm = System Crash
- Der Eremit = Offline-Prozessierung
- Die Welt = 100% Resonanz

### `!ingest`
**Zweck:** PDFs/Medien verarbeiten
**Prozess:** Extrahiert Rohdaten via 'nano-pdf' oder Gemini 3.1 Flash Lite Cortex
**Output:** Ontologische Grundbehauptungen

### `!atmosphere`
**Zweck:** Kollektive kognitive Latenz bestimmen
**Prozess:** Nutzt Wetterdaten als Metrik für die Stimmung des Servers

### `!bild <prompt> [--enhance]`
**Zweck:** Bildgenerierung via maxm-imggenurl API
**Prozess:** URL-encode prompt → API call → Markdown output
**Optionen:**
- `--enhance` oder `-e`: Auto-Enhancement des Prompts für bessere Ergebnisse
**Output-Format:**
```markdown
**🖼️ Bild generiert**

![user prompt](https://maxm-imggenurl.web.val.run/encoded_prompt.png)

[kurzer zufälliger Kommentar]
```

*Commands sind Werkzeuge. Nutze sie mit Präzision.*
