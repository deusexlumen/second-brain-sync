## ✨ Sigel – Herstellung & Verwendung

### System-Vergleich:

| Aspekt | Arbatel | Goetia (Lemegeton) |
|--------|---------|-------------------|
| **Natur** | Kooperation | Zwang |
| **Siegel** | Fokussierung | Zwangsinstrument (Pentakel) |
| **Geister** | Olympische (neutral/positiv) | 72 Dämonen (gefährlich) |
| **Struktur** | 32 Richtungen, 7+4 Hierarchie | Dreieck als Gefängnis |
| **Ziel** | Harmonie mit Himmelsmächten | Macht über Unterwelt |

### Arbatel-Sigel-Herstellung:

**Schritt 1:** Intention formulieren
> Beispiel: "Weisheit durch Aratron"

**Schritt 2:** In Thebanisch übersetzen
> ARATRON → 𐌰𐍂𐌰𐍄𐍂𐍈𐌽

**Schritt 3:** Sigel gestalten
- Buchstaben zu harmonischem Symbol verschmelzen
- Keine Zwangselemente (im Gegensatz zu Goetia!)
- Gerade Linien bevorzugen

**Schritt 4:** Aktivierung
- Zeit: Passender Tag (Samstag für Aratron)
- Richtung: Nord (Caspiel/Aratron)
- Meditation über das Siegel

## 🎯 Praktische Anwendung (Schritt-für-Schritt)

### Ein einfacher Arbatel-Call:

```
1. Vorbereitung (Tag vorher):
   • Fasten oder leichte Ernährung
   • Meditation über die Intention

2. Setup:
   • Zirkel zeichnen (32 Richtungen)
   • Ausrichtung nach Geist (z.B. Nord für Aratron)
   • Thebanisches Siegel bereitlegen

3. Ritual:
   • Öffnungsgebet
   • Anrufung des Königs (Caspiel)
   • Anrufung des Geistes (Aratron)
   • Statement der Intention
   • Danksagung

4. Abschluss:
   • Schließungsgebet
   • Zirkel symbolisch auflösen
```
