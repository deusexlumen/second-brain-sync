# Hoppe — Sozialismus vs. Kapitalismus: Die zwei Archetypen der Wirtschaftsordnung (1989)

## Dekonstruierter Kern-Gedanke
Hoppe reduziert alle Wirtschaftssysteme auf zwei Archetypen: **Kapitalismus** (reiner Schutz von Privateigentum, freie Assoziation, keine Ausnahmen) und **Sozialismus** (jede Abweichung vom kapitalistischen Ideal). Er identifiziert vier Sozialismus-Formen: russischer Stil (Vergesellschaftung), sozialdemokratischer Stil (Umverteilung), konservativer Stil (Preiskontrollen, Regulierungen, Verhaltenskontrollen), und technokratischer Stil (stückweise Intervention). Der entscheidende Mechanismus: **Je mehr Eigentumsrechte verletzt werden, desto ärmer die Gesellschaft**. Konservatismus ist eine besonders verdeckte Form des Sozialismus — er nutzt Regulierungen statt direkter Vergesellschaftung, um den Status quo zu zementieren.

## Ariadne-Brücken (Cross-Domain Links)
- **Isomorphe Struktur:** [[fodor-modularity-1983]] -> *Mechanik:* Fodor trennt Input-Systeme (modular, gekapselt, schnell) von Zentralen Systemen (holistisch, querational, langsam). Hoppe trennt Kapitalismus (modulare Ökonomie: domain-spezifische Eigentümer, gekapselte Entscheidungen, schnelle Marktpreise) von Sozialismus (zentrale Ökonomie: holistische Planung, querationale Umverteilung, langsame Bürokratie). Ist Kapitalismus eine **modulare Gesellschaftsarchitektur** — und Sozialismus ein Versuch, modulare Systeme durch zentrale Kontrolle zu ersetzen, mit denselben Effizienzverlusten, die Fodor für Zentrale Systeme vorhersagt?
- **Spannungsfeld / Paradoxon:** [[greene-oberflaeche-menschlicher-natur]] -> *Konflikt:* Greene's "Gesetz des zwanghaften Verhaltens" besagt: Charakter ist das, was Menschen wiederholt tun — nicht was sie sagen. Hoppe's Analyse zeigt, dass Sozialisten **wiederholt Eigentumsrechte verletzen** (was sie tun), während sie **Eigentumsrechte rhetorisch verteidigen** (was sie sagen). Ist der Sozialismus ein **Narzissmus auf kollektiver Ebene** — eine kognitive Dissonanz zwischen Selbstbild ("Wir helfen") und Verhalten ("Wir nehmen")? Und könnte Greene's Methode (Musterbeobachtung statt Wort-Interpretation) die Effizienz der Hoppe'schen Diagnose erhöhen?

## Synthetischer Impuls (Emergenz)
> Wenn wir Hoppe's Sozialismus-Typologie als **Taxonomie von Governance-Bugs** modellieren — in dem jede Sozialismus-Form eine spezifische Verletzung des Eigentums-Moduls ist — könnten wir dann **Patches** entwerfen? Russischer Sozialismus = "Total Ownership Override" → Patch: Decentralisierung. Sozialdemokratie = "Progressive Memory Leak" (Umverteilung entfernt Ressourcen aus Produktion) → Patch: Flat Tax. Konservatismus = "Legacy Code Freeze" (Status quo zementieren) → Patch: Dynamische Regulierung. Ist politische Philosophie ein **Software-Engineering-Problem**?
