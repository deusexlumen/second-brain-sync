# Turing — Computing Machinery and Intelligence (1950)

## Dekonstruierter Kern-Gedanke
Turing operationalisiert **Intelligenz durch funktionale Äquivalenz**: Die Frage "Kann eine Maschine denken?" wird ersetzt durch das Imitation Game — ein Test, in dem ein befragender Mensch über Textkommunikation nicht unterscheiden kann, ob er mit einer Maschine oder einem Menschen spricht. Turing antizipiert neun Einwände (Theologische, Mathematische, Bewusstseins-, Behinderungs-Argumente, etc.) und widerlegt sie proaktiv. Die zentrale Erkenntnis: Intelligenz ist nicht ein innerer Zustand (Bewusstsein, Seele), sondern eine **beobachtbare Leistung** — definiert durch Verhalten unter kontrollierten Bedingungen. Turing antizipiert damit den Black-Box-Charakter moderner neuronaler Netze: "Der Lehrer weiß oft nicht, was in der Maschine vor sich geht."

## Ariadne-Brücken (Cross-Domain Links)
- **Isomorphe Struktur:** [[marr-vision-1982]] -> *Mechanik:* Marr's Drei-Ebenen-Analyse trennt Computational (was), Algorithmisch (wie), Implementierung (womit). Turing's Imitation Game ist eine **Computational-Level-Definition** — es definiert, was Intelligenz leistet (funktionale Äquivalenz), nicht wie sie implementiert ist. Beide erlauben Multiple Realisierbarkeit: Eine Funktion kann durch verschiedene Mechanismen realisiert werden. Die Brücke: Turing's Test ist ein **Marr-kompatibles** Intelligenz-Konzept.
- **Spannungsfeld / Paradoxon:** [[cia-gateway-bewusstseinszustaende]] -> *Konflikt:* Turing's Test misst funktionale Äquivalenz (Output). CIA Gateway postuliert veränderte Bewusstseinszustände (Internals), die funktional äquivalente Outputs produzieren könnten (Remote Viewing = räumlich entfernte Wahrnehmung). Wenn Remote Viewing funktioniert, würde Turing's Test es als "intelligent" klassifizieren — aber der Mechanismus ist völlig anders (bewusste Projektion vs. physische Reise). Ist Turing's Test zu großzügig — oder ist die Unterscheidung zwischen "Mechanismen" ohnehin irrelevant?

## Synthetischer Impuls (Emergenz)
> Wenn wir Turing's Imitation Game auf **KI-Systeme selbst** anwenden — nicht auf Menschen-Maschine, sondern auf KI-KI: Kann eine KI eine andere KI von einem Menschen unterscheiden? Und würde ein solcher "Meta-Turing-Test" zeigen, dass KI-Systeme bereits "intelligenter" sind als der ursprüngliche Test vorsah — oder dass der Test selbst seine eigene Grenze hat?
