## 🎯 Was ist das?

**Second Brain Sync** ist ein **automatisiertes Wissensmanagement-System** — inspiriert von Niklas Luhmanns Zettelkasten, aber für das 21. Jahrhundert.

Statt Karteikästen: **KI-gestützte Verarbeitung**.  
Statt manuellem Verknüpfen: **Autonome Strukturerkennung**.

### Die Philosophie

```
Input → KI-Verarbeitung → /RAW → Strukturierung → Emergentes Wissen
```

- **Keine Ordner-Hierarchie** — nur Verknüpfungen
- **Kein Vergessen** — alles persistiert
- **Keine Isolation** — Ideen finden sich selbst

### Die `/RAW`-Philosophie

> **ROH ist heilig.**

Jede Information landet erst in `/RAW` — ungefiltert, unverarbeitet, authentisch.  
Erst dann beginnt die KI-gestützte Strukturierung.

Das ist **nicht ein Archiv**. Das ist ein **Brutkasten** für Ideen.
